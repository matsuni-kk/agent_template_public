# Post Deploy Cleanup チェックリスト

## 削除前確認

### デプロイ完了確認
- [ ] GitHubリポジトリが作成されている
- [ ] リポジトリにpush済み
- [ ] リポジトリURLが記録されている

### バックアップ確認
- [ ] 重要なファイルはGitHubリポジトリに含まれている
- [ ] Stock/に保存すべき成果物は移動済み
- [ ] 未コミットの変更がないことを確認

### ユーザー確認
- [ ] 削除対象をユーザーに提示
- [ ] ユーザーから削除許可を取得

## 削除対象

### 必須削除
- [ ] `output/{agent_name}/` - デプロイ済みエージェント

### 任意削除
- [ ] `Flow/{year_month}/{date}/{agent_name}/` - 作業ファイル
- [ ] 空の `Flow/` サブディレクトリ

## 削除後確認

- [ ] `output/{agent_name}/` が存在しない
- [ ] 関連する `Flow/` ディレクトリが存在しない
- [ ] 他のエージェントに影響がない
