# Agent Discovery チェックリスト

## Phase 1: 情報収集
- [ ] ドメイン名が確定（スネークケース）
- [ ] エージェント表示名が確定
- [ ] 解決課題が明確化
- [ ] 対象ユーザーが特定
- [ ] 必須機能が3つ以上列挙

## Phase 2: 業界調査
- [ ] 業界標準フレームワークを調査（BABOK、AMA等）
- [ ] Web検索で最新トレンドを確認
- [ ] 出典URL・取得日時を記録
- [ ] framework_research.md を作成

## Phase 3: 要件文書化
- [ ] draft_requirements.md を作成
- [ ] 不確定事項をTODO形式で列挙
- [ ] 合意時刻を明記

## Phase 4: 構造承認
- [ ] Flow / Stock / Archived 構成を説明
- [ ] ユーザー承認を取得
- [ ] 承認内容をタスクリストに記録

## Phase 5: タスクリスト生成
- [ ] {{today}}_agent_creation_tasklist.md を生成
- [ ] framework_research_done=true を記録
- [ ] 次アクションを明記

## 承認必須項目
**以下の承認が得られるまで後続工程へ進まない：**
- エージェント要件への承認
- フォルダ構成への承認
- 成功基準への合意
